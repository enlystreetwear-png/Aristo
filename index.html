⚙️ One-time Firebase Setup

This app needs Firebase to sync data across all devices. Follow the steps below (takes ~5 minutes, completely free).

1Create Firebase project

Go to console.firebase.google.com → Click "Add project" → Name it (e.g. expense-manager) → Disable Google Analytics → Click "Create project"

2Enable Firestore Database

In your project → Left sidebar → "Firestore Database" → "Create database" → Choose "Start in test mode" → Select any location → Click "Enable"

3Get your config keys

In Firebase → Click ⚙️ gear icon → "Project settings" → Scroll down → Under "Your apps" click "</> Web" → Register app → Copy the firebaseConfig values below:
